Overview

This project comprises a set of Python scripts designed to interact with the Bot Hosting service provided by bot-hosting.net. The scripts are intended to offer users a convenient command-line interface for managing their Bot Hosting accounts and servers.

Features

Account Management

  • Retrieve account information, including username, ID, coins, and avatar.
  • Check the total amount of coins in the account.
  • Obtain affiliate data, including coins per referral, referral link, and usage statistics.
  • Generate a new SFTP password for added security.

Server Management

  • Retrieve information about servers, including renewal details, plan specifications, and status.
  • Change the programming language of a server, supporting languages like Java, Python, Node.js, Lua, Deno, and Nodemon.
  • Delete servers with user confirmation.
  • Display a list of all servers associated with the account.

Usage

To use the provided scripts, users need to obtain their authentication ID (auth id) from the Bot Hosting website and incorporate it into the scripts.

# Initialize the Account and Server classes
acc = Account(auth_id)
serv = Server(auth_id)

# Example usage of the functions
serv.delete()  # Delete a server (user confirmation required)
serv.show()    # Display a list of all servers
serv.get_info()  # Retrieve detailed information about a specific server

acc.coins_amount()  # Check the total amount of coins in the account
acc.about()  # Display a quick overview of the account
acc.id_check()  # Check if the Auth ID is valid
acc.sftp_pass()  # Generate a new SFTP password

How to Get Started

  1. Obtain your authentication ID (auth id) by following the instructions provided in the get_auth_id method of the Account class.
  2. Initialize the Account and Server classes with your authentication ID.
  3. Use the provided methods to manage your Bot Hosting account and servers.

Feel free to explore and customize the scripts based on your specific needs.
